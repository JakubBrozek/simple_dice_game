# simple_dice_game
Simple game made in python as an exercise

Rules:
1. There are 8 rounds.
2. You and the enemy take turns rolling a d6.
3. On your move you can choose to either 'roll' the dice or 'pass'.
4. Every roll value is added to score in current round, if you roll '1' you lose all progress.
5. If you choose to 'pass' round ends and your 'round score' is added to your total points.
6. Enemy plays the same way.
7. Whoever has more poinst after 8th round wins
